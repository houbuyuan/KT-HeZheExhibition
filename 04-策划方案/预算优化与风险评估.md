# HeZhe Exhibition (同江印象·赫哲族文化巡展) 预算优化与风险评估

> [!IMPORTANT]
> **预算总额：500,000 RMB (50万)**
> 考虑到 300-500 平方米的展陈面积，此预算属于**极度紧张**级别（均价仅约 1,000-1,600 元/平米）。
> 为了确保效果，必须对高成本数字化项进行严格筛选或方案降级。

## 1. 高成本风险项分析 (Red Flags)

以下展项若按“标准版”执行，极有可能导致预算超支：

| 风险项 | 预估成本 (标准版) | 成本构成分析 | 降级建议 (Economy Mode) |
| :--- | :--- | :--- | :--- |
| **三江汇流互动投影** | 15.0 - 25.0 万 | 8台激光投影机 + Lidar雷达 + 定制流体互动引擎 | 减少投影叠加面积，改用4台投影；或取消实时物理追踪，改用固定感应触发动画。 |
| **无限镜面全息序厅** | 8.0 - 12.0 万 | 大面积高品质单向透视镜 + 全息成像结构 + 镜面基础装修 | 减小镜面覆盖面；改用高反射成品镜面贴膜；全息改用“伪全息”投影膜。 |
| **定制沉浸式内容开发** | 10.0 - 20.0 万 | 神话/渔猎/史诗三部定制数字影片的渲染与CG | 整合数字内容，部分采用高清实拍素材（航拍）进行剪辑，减少三维建模量。 |
| **AR互动与任务系统** | 5.0 - 8.0 万 | 定制App/小程序开发 + 后台管理系统 + AR识别算法 | 采用基于微信的轻量扫码互动（无AR）；或利用现成的H5插件。 |

## 2. 预算分配方案 (建议)

针对 50 万总预算，建议采取 **“重氛围、轻交互、模块化”** 的策略：

| 类别 | 建议预算 | 策略 |
| :--- | :--- | :--- |
| **空间装修与基础照明** | 15 万 | 采用模块化展墙、轻质桁架结构，大幅减少传统固定装修。 |
| **硬件设备租赁/采购** | 15 万 | 核心区使用 4-6 台投影机。部分设备可考虑二手或租赁以降低成本。 |
| **数字内容与软件开发** | 10 万 | 寻找高性价比团队，利用现有文化素材进行二次创作，避免从零建模。 |
| **展项制作与互动装置** | 5 万 | 开发 1-2 个核心互动点（如体感试穿），其余部分采用静态陈列配合导览信息。 |
| **运输与巡展运营预留** | 5 万 | 考虑到是巡展，必须保留运输和拆装的备用金。 |

## 3. 核心降级策略

1.  **以“光影”代“交互”**：减少雷达追踪等复杂反馈逻辑，通过环境光影循环播放打造氛围，通过手机小程序扫码完成信息交互。
2.  **以“实物”衬“数字”**：利用现有的非遗实物（鱼皮衣、鱼叉、独木舟）作为视觉主体，数字投影仅作为背景点缀，而不是主展项。
3.  **结构轻量化**：巡展需要快拆、快装。采用织物喷绘背景+投射照明，效果优于实景装修且成本极低。
