# HeZhe Exhibition (同江印象·赫哲族文化巡展) 技术与互动方案

> **技术原则**：各展项采用“**独立节点、中控解耦**”的架构。每个展项均配有独立的计算单元（如迷你工作站、控制器），确保在重新组合或单个模块故障时，不影响全局展示。

## 1. 核心技术应用

### 1.1 序厅：创世弧光 (The Genesis Arc)
- **技术**：小微型弧形金属软幕（约 3-5米弧长）+ 短焦激光投影机 x 2（融合）。
- **[预算优化]**：小尺寸异形幕 + 2台投影，造价远低于全厅沉浸式，且内容制作量小而精。
- **视觉逻辑**：打造“半私密包裹感”。弧幕视角覆盖观众前方 120-160度，刚好形成沉浸视野，不干扰外部空间。
- **互动**：**雷达触碰（可选）**。观众触碰画面中的“神火”，火光顺着弧幕边缘流淌。

### 1.2 第一单元：沉浸式三江汇流
- **技术**：L型折幕投影（纵向墙面 + 地面流动投影）。
- **[预算预警]**：高亮度投影机集群与实时流体互动的开发费用可能占据预算的40%以上，属于核心高能耗项。
- **感官体验**：结合流体动力学模拟水流效果。地面采用压力传感器或雷达追踪（Lidar），观众走过时，水流会绕开脚部并溅起浪花。
- **音频**：超指向性音箱（扬声器阵列），模拟局部水流声。

### 1.3 第二单元：AR鱼皮技艺解密
- **技术**：AR（增强现实）透镜 / iPad 互动。
- **体验流程**：观众将设备对准实物展柜中的鱼皮衣，屏幕上会叠加上数字化细节（分层展示内部构造、缝制路径）。
- **数字化试穿**：体感Kinect装置，识别观众体态，将虚拟鱼皮衣实时“穿”在观众身上。

### 1.4 第三单元：史诗剧场
- **技术**：7.1声道环绕声场 + 动态激光投影。
- **内容控制**：中控系统同步光影、声音与气味系统（模拟森林泥土气息）。

### 1.5 辅助展项技术详情
- **溯源万花筒**：
    - **技术**：物理光学结构 + LED背光模组。
    - **特点**：无需电力驱动内容，通过更换内部透光胶片实现内容更新，维护成本极低。
- **“岁时之歌”二十四节气墙**：
    - **硬件**：象牙白陶瓷喇叭 x 12、激光切割PVC造型板 x 1组（多层叠加）、金色UV打印/即时贴 x 24、音频主板 x 12。
    - **软件**：嵌入式音频循环播放程序。
- **乌日贡喜悦之柱 (The Pillar of Joy)**：
    - **硬件**：手工雕刻木质图腾柱（内含钢结构） x 1、32寸竖向高亮电容触摸屏 x 1、地面虚拟篝火投影灯 x 1、装饰彩带/铜铃 x 1套。
    - **软件**：乌日贡大会多媒体互动系统（包含叉草球、天鹅舞等专题内容）。
- **萨满拍照互动系统**：
    - **硬件**：高清工业摄像头 (USB 3.0) + 55寸竖屏显示器。
    - **技术**：基于深度学习的人脸/人体关键点检测（AR Try-on）。
    - **软件**：内置多套萨满服饰 3D 模型，支持实时缩放与跟随，支持二维码分享海报。

“探秘巡礼”是一个贯穿全场的数字化互动系统，通过“游戏化”的逻辑，让原本沉浸但碎片化的视觉内容串联成一段有意义的“文明收集”之旅。

### 2.1 核心流程 (UX Flow)
1.  **入场启动**：观众扫描序厅入口处的“神谕之门”二维码，进入微信小程序，领取“赫哲族探秘护照”。
2.  **碎片收集**：
    - **一单元 (水之印)**：在三江汇流投影区，寻找隐藏在水流中的二维码，打卡解锁“三江地缘记忆”。
    - **二单元 (鳞之彩)**：使用 AR 透镜对准指定的鱼皮衣展柜，不仅能看分解图，点击屏幕还能“采撷”一个鱼皮纹样。
    - **三单元 (歌之魂)**：沉浸式剧场结束后，扫码完成一个小型的“伊玛堪曲调辨析”挑战，收集声音碎片。
3.  **最终进化**：集齐所有碎片后，系统会自动合成一张属于该观众的、独一无二的“非遗守护者”数字海报。

### 2.2 技术实现逻辑
- **识别技术**：
    - **轻量级**：普通二维码（QR Code）用于快速打卡，确保低性能手机也能参与。
    - **沉浸式**：图像识别（Image Recognition），AR 引擎识别鱼皮衣上的特定纹样作为触发源。
- **数据管理**：
    - 采用云端数据库记录每个微信 OpenID 的进度。
    - 积分墙逻辑：每完成一个任务获得相应的“非遗积分”，可在出口兑换电子勋章。

### 2.3 AR 触发详情
- **类型**：SLAM（即时定位与地图构建）与空间锚点。
- **互动场景**：在二单元，观众站在指定脚印位置，手机或AR透镜中会浮现一名“虚拟传承人”，演示如何捶打鱼皮。
